Danes je v uporabi ogromno različnih programskih jezikov. V nadaljevanju sledi opis 5ih najbolj priljubljenih.

Python

Python je visokonivojski programski jezik, ki je bil razvit leta 1991. Razvil ga je Guido van Rossum, ki je bil velik oboževalec angleške humoristične skupine Monthy Python in od tod sledi ime tega programskega jezika. Glavni namen njegovega razvoja je bil ta, da spodbudi čim več začetnikov k programiranju, saj je zelo preprost za učenje in uporabo.

Python velja za priljubljen jezik zaradi njegove enostavnosti ter jasne in razumljive sintakse. Prav to ga dela tako enostavnega za uporabo. Njegova prednost je zelo velika knjižnica, ki pripomore k lažjemu, hitrejšemu in učinkovitejšemu programiranju.

Seveda pa ima v primerjavi z drugimi programskimi jeziki tudi svoje slabosti. Python je veliko počasnejši od katerih drugih programskih jezikov kot sta na primer C ali C++.

Povzeto iz Python Software Foundation. (n.d.). About Python.

JavaScript

JavaScript je visokonivojski programski jezik, ki je namenjen predvsem izdelovanju interaktivnih spletnih strani. Ustvarjen je bil leta 1995 pri podjetju Netscape, danes pa je temeljni del spletnega razvoja skupaj s HTML in CSS.

Prednost JavaScripta je ta, da se izvaja direktno v brskalniku, kar je dobro za ustvarjanje dinamičnih in interaktivnih spletnih strani za katere je bil ustvarjen. Prav tako kot pri Pythonu se lahko pri JaviScriptu zanesemo na ogromno knjižnico, ki je že vgrajena v jezik

Kot slabost JaveScripta pa lahko izpostavimo spletno varnost, saj se izvaja direktno v brskalniku in je tako izpostavljen različnim napadom. Prav tako ni primeren za kakšna podjetja, ki imajo opravka z občutljivimi podatki, saj so ti vidni vsem uporabnikom.

Povzeto iz Mozilla Developer Network. (n.d.). JavaScript Guide

Java

Java velja za visokonivojski jezik, ki je bil razvit leta 1995 s strani podjetja Sun Microsystems. Java slovi po sloganu "zapiši enkrat, zaženi povsod", kar pomeni, da lahko programe, ki so bili napisani v Javi zaženemo na kateremkoli sistemu z nameščenim Java Virtual Machine (brez JVM programi zapisani v Javi ne delujejo)

Verjetno je največja prednost Jave že prej omenjeni JVM, ki poskrbi, da se isti porgram izvaja na kateremkoli operacijskem sistemu. Javina stroga sintaksa nam omogoča hitro odkrivanje napak in nam olajša programiranje.Velik plus v današnjih časih pa gotovo predstavlja dejstvo, da je ta programski jezik zelo primeren za ustvarjanje spletnih in mobilnih aplikacij.

Kljub temu, da imata Java in JavaScript podobna imena je potrebno poudariti, da jih ne smemo enačiti med seboj, saj gre za dva popolnoma različna programa.

Povzeto iz Oracle. (n.d.). Java Technologies.

C

Za razliko od vseh prej omenjenih programskih jezikov je C nizkonivojski jezik, ki je bil razvit v začetku 70-ih let prejšnjega stoletja. Ustvarjen je bil predvsem za razvoj sistemskih programov, kot so gonilniki in operacijski sistemi.

C je eden izmed najbolj priljubljenih programskih jezikov zaradi svoje hitrosti in učinkovitosti. Prav tako je zelo kompakten in zanesljiv, potrebno pa je poudariti, da je C tudi osnova za nekatere druge jezike, kot npr. Java

Seveda pa ima C v primerjavi z ostalimi programskimi jeziki tudi svoje slabosti. Gre razmeroma star programski jezik, zato ima kar nekaj pomankljivosti v primerjavi z novejšimi. V oči predvsem bode dejstvo, da mora programer ročno upravljati s pomnilnikom, kar pogosto vodi do napak.

Povzeto iz GeeksforGeeks. (n.d). C Language Introduction.

Swift

Swift je sodoben programski jezik, ki ga je razvilo podjetje Apple leta 2014. Pomemben je zato ker je vodilni programski jezik za sistem iOS, torej za naprave kot so iPhone, iPad, Mac, Apple Watch...

Swift je hiter in učinkovit, saj je zasnovan za visoko zmogljivost. Poleg tega pa je tudi zelo enostaven za uporabo, ker je lahko berljiv, z jasnim in jedrnatim sintaksnim slogom.

Swift je primarno namenjen upori na napravah proizvajalca Apple. Kljub temu obstaja podpora za Linux in Windows, vendar se tam slabše obnaša, kar lahko vzamemo za slabost. Prav tako ima veliko manj knjižnjic in orodij sploh v primerjavi s katerimi bolj uveljavnimi jeziki, kot je recimo Python.

Povzeto iz oursera. (n.d.). Programming in Swift.
